# Mini-PhoneBook (C Project)

Mini PhoneBook is a simple console-based application written in C.
It allows users to store, display, and manage a small list of contacts directly from the terminal.

This project is designed for beginners and focuses on mastering fundamental C concepts such as:

- Low-level input/output using write and read
- Loops and conditionals
- Arrays and basic data storage
- Code organization and logic building

Features:
 - Add a new contact (name + phone number)
 - Display all saved contacts
 - Search for a contact by name (optional / upcoming)
 - Exit the program
